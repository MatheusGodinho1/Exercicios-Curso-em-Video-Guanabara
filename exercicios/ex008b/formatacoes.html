Todos os tipos de formatações semanticas em HTML

Negrito

Para adicionarmos um texto em destaque, ou negrito usamos a tag <strong>

Itálico

Para darmos ênfase em um texto usamos a tag <em>

Marcado

Para adicionarmos uma marcação em um texto usamos a tag <mark>

Deletado/Taxado

Para mostrarmos que um texto foi delegado ou que não deve ser levado em consideração pelo leitor usamos a tag <del>

Inserido

para mostrarmos que um texto foi adicionado recentemente usamos a tag <ins>

Sobre Escrito

Usamos a tag <sup> quando precisamos deixar um texto SOBRE ESCRITO para escrever por exemplo m2 ou x2

Sub Escrito

Usamos a tag <sub> para deixar um texto SUB ESCRITO, isso ajuda a escrever palavras como h2O
