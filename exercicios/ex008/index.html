Tipos de formatações

Negrito semantico

Isso é um texto em NEGRITO semantico

Itálico Semantico

Isso é um texto em ITALICO ou ênfase

Marca texto

Isso é um texto MARCADO

Texto Delegado

para mostrar o texto taxado ou deletado, usamos a tag del que significa que o texto está ali, pode até ser lido, mas deve ser desconsiderado pelo leitor. Exatamente como eu acabei de fazer.

Texto inserido

Nesse caso, se colocarmos um texto qualquer dentro de ins, estamos dizendo que o texto está ali, deve ser lido e você deve prestar atenção nele.

Texto sobre escrito

usamos textos sobre escrito com a tag <sup> para escrever coisas tipo: x2 ou metro2

Texto subescrito

usamos textos subescritos com a tag <sub> para esrever h20 por exemplo.
